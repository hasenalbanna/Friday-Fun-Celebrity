# Friday Fun Celebrity Mobile

This is an Expo app scaffold for Android and iOS.

## What it includes

- Room key creation and room joining
- Local room storage with AsyncStorage
- Image upload into a room from the device gallery
- A rooms section that shows the saved rooms on the device

## Run locally

1. Install dependencies in this folder.
2. Start the app with `npm run start`.
3. Use Expo Go for testing during development.

## Play Store path

1. Replace the placeholder package name in `app.json` if needed.
2. Add app icons and splash assets.
3. Run an Android production build with EAS.
4. Upload the generated `.aab` to Google Play Console.
